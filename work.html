In March 2021, I started learning HTML and CSS on Codecademy. The initial goal was to improve my copywriting skills.

This is the very first project I created to practise HTML and CSS. It’s a form(what a surprise!) for my blog “A Reel Trip.”

To complete the “Responsive Web Design Certification” on freeCodeCamp, I had to submit a Product Landing Page. So, I thought I’d get creative and sell some Jojoba Oil. With the help of a YouTube tutorial, I built this product landing page.

It made sense at some point to learn how to use Bootstrap 5. FreeCodeCamp’s Front End Development Libraries Certification includes a Bootstrap Course. It was a good starting point. Did I create a project to practise Bootstrap? Yes. Am I proud of it? Not really. But at least I think that I made good use of Bootstrap for my tribute page.

Here comes my first actual attempt to put into practice some basic Javascript knowledge. It’s a word count tool. A character count for Twitter and LinkedIn, which could be used by social media content writers, was added as well. Try it out!.

While I was learning a lot and getting deeper into Javascript, my HTML and CSS were getting a bit rusty, so I built this restaurant website to get back on track.

The FAC Weekly Prompts were a big part of my learning journey. In Week 5, I worked on the Codename Validator prompt. To practice working with arrays and array methods in Week 6, I created this page, where you can "play with numbers."

In Week 6, I created a project gallery that uses a flexbox layout. It was a good opportunity to use some cool CSS animations as well.

Week 7 was all about creating a feature for our Application Website, so that’s when I came up with the idea of the quiz for this website.

The object prompts helped me get familiar with object methods and the importance of different data types to convert one data type to another. Here's the repository of my object practice project.

This website feels like the “official” beginning of a new, exciting journey. But until it starts, here’s the journey of this website on Github.
